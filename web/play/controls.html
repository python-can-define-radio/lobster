LOBSTER

Instructions:

  • Check incoming messages using the envelope icon for mission instructions.
  • Find the transmitter using Lines of Bearing.
  • Enter your 10 digit grid coordinates in the box provided.
  • Hit the 'Enter' key or click Submit button.

Controls:

  • 'W', 'A', 'S', 'D' to move
  • 'Shift' to run
  • 'G' to toggle LOB collection
  • 'C' to clear LOB's
  • Zoom using the + and - magnifying glass
